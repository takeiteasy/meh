# meh
feh for OS X, inspired by [DrabWeb/macfeh](https://github.com/DrabWeb/macfeh)

See top of ```main.m``` for TODO list

## Shortcuts
```
ESC, Q = Close window
CMD+Q = Close all windows
J, L Arrow, D Arrow = Prev image
K, R Arrow, U Arrow = Next image
O = Open file dialog
```
