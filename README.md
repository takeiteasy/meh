# meh
feh for mac, inspired by [DrabWeb/macfeh](https://github.com/DrabWeb/macfeh)

### Work in progress, not nearly completed

See top of ```main.c``` for TODO list

## Shortcuts
```
ESC, Q = Close window
CMD+Q  = Close all windows
J      = Prev img
K      = Next img
```
